Office Game

In this game you need to survive as long as possible.
The main goal is to archive or sort out the files.
In the office room on the computer you can put the file you just picked up in the right folder.
All 3 lives gone then its game over

If the game starts you begin in the conveyer room.
First you always start with a TXT File to deliver and you start with a quota of 30 points in 3 minutes.
If you deliver the file in the right folder you get 10 points.
Every 10-15 seconds a new file will be spawning.
There is also a limit of 4 files to have on the coneyer.


This is the computer where you sort out your files and there is a shop wherer eyou can upgrade your characterstats forexample: speed
Here you see a WAV File that is gonna placed in the audio file.
That is the correct folder, if it was a different folder you fail.
If you fail 3 times 1 live goes down but you can always buy a fail in the shop for 5 coins.
For every file you succesful deliver you get 5 points and 1 coin.

Here you see an event happening in the game and for every event you get 2 points and 5 coins.
This event is dry plants so by filling the watering can with water and then watering the plant you succesfull complete the event.
There is also a fire alarm event(if there is a fire in a rooom you pick up the extinguisher in the equipment room and put out the fire).
And a virus event (If there is a virus you need to check every folder in the computer and click on the Virus.EXE to delete it).
